# Graph Coloring Lab

Interactive graph coloring solver using depth-first backtracking and degree-based node ordering.

## Highlights
- Random graph generation with tunable node count and edge density.
- Live solving animation for NP-complete graph coloring.
- Constraint-check and backtrack telemetry.
- Conflict highlighting for invalid assignments.

## Run
Open `index.html` in a modern browser.
